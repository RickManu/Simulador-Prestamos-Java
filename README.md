#  Simulador de Créditos 

Simulador financiero desarrollado en **Java Swing** que permite calcular tablas de amortización con cuota nivelada.

###  Lo que hace este proyecto:
- **Cálculo Preciso:** Desglose de Interés, Capital y Saldo mensual.
- **Formato Contable:** Alineación a la izquierda con signo de Quetzal (Q) y comas de miles.
- **Validación de Datos:** Evita errores si dejas campos vacíos o usas letras.
- **Interfaz Nimbus:** Un diseño más moderno y limpio que el estándar de Java.

###  Tecnologías usadas:
- Java JDK 17 / Maven
- NetBeans IDE 22
- Git & GitHub para control de versiones
